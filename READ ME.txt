Notre projet repose sur deux sources de données, 
l'une recense les accidents de voiture ayant eu lieu au Royaume-Uni en 2021 et 2022
(raw/Accident/Road Accident Data.csv)
et l'autre se base sur les positions géographiques et la date des accidents 
et ajoute les données météorologiques de chaque accident
(raw/Weather/weather_data2.json)

-dans le dossier raw, vous trouverez le csv original extrait de Kaggle.
Dans le dossier 'Weather' vous trouverez les requêtes vers l'API Open Data Meteo et 
le json qui en découle.

Dans le dossier 'Accident', le code qui permet de reformater les données originales
de Kaggle


Dans le dossier 'data fusion', se situent le code de fusion des deux bases ainsi 
que le  fichier csv 'fusion.csv' qui en est issu.

Enfin dans le dossier 'analyse', vous trouverez tout d'abord les données reformatées 
pour qu'elles soient propices à une régression ainsi que le code qui permet ce 
reformatage. Puis le code d'analyse des données, avec en variable expliquée, 
la gravité de l'accident. Puis des tests de robustesse, dans lesquels nous appliquons 
les mêmes modèles mais en ayant en variable expliquée le nombre de victimes et 
le nombre de voitures impliquées.
